# Primitive Type: `u16`
This is a short form data type or representing a 16bit unsigned integer.
The maximum value of this data type is 65535 and the minimum value is 0.

# Example
```cpp
int main() {
    u16 number = 65535;
    printf("The number is %d\n", number);

    return 0;
}
```
