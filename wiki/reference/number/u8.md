# Primitive Type: `u8`
This is a short form data type or representing a 8bit unsigned integer.
The maximum value of this data type is 255 and the minimum value is 0.

# Example
```cpp
int main() {
    u8 number = 255;
    printf("The number is %d\n", number);

    return 0;
}
```
