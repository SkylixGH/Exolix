# Primitive Type: `u32`
This is a short form data type or representing a 32bit unsigned integer.
The maximum value of this data type is 4294967295 and the minimum value is 0.

# Example
```cpp
int main() {
    u32 number = 4294967295;
    printf("The number is %d\n", number);

    return 0;
}
```
